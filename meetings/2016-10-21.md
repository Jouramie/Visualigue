#### Comment enregistrer le déplacement d'un joueur :

1. Appuyer sur le bouton record
2. Appuyer avec le bouton gauche sur le joueur. 
    1. Bind la position de la sourie avec celle de l'élément mobile.
2. À chaque tick, créer un keyframe dans la trajectoire de l'élément mobile.
3. Lâcher le bouton gauche de la sourie. 
    1. Unbind la position de la sourie avec celle de l'élément mobile.
    
    
#### Comment updater le front-end avec le back-end :

1. Click sur l'outil d'ajout d'élément
    1. onActionXElement(e) recoit l'évènement
    2. index = e.getSource().getIndex() // Retourne le l'index de l'élément sur lequel on a cliqué dans le sous-menu
    3. nomElementDescription = XElementImages.get(index)[0]
    4. controller.selectElementDescription(nomElementDescription)
2. Click dans la scène -> pouf!
    1. onMouseClick(e) recoit l'évènement
    2. controller.addElement(pos)
        1. selectedElement = new Element(pos)
    3. elemListener = new UIXElement() // UIXElement implement elementListener
    4. controller.subscribeElementListener(ID, elemListener)
        1. strategy.getElement(ID).setElementListener(elemListener)
